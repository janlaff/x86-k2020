# x86-k2020
32 Bit Mini Operating System

![alt text](/images/kernel01.png)

## Dependencies
i686 gcc compiler
nasm
qemu

## Build
```
$ ./build.sh
```

## Run (Debug)
```
$ ./qemu.sh
```
